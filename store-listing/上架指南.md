# SAT APP 应用商店上架指南

## 一、APK 构建

### Debug 版（微信/QQ 分享用）
```bash
bash build-apk.sh debug
```
输出: `output/SAT-debug.apk`
- Debug APK 可直接安装使用
- 用户需要允许"安装未知来源应用"

### Release 版（应用商店上架用）
```bash
bash build-apk.sh release
```
输出: `output/SAT-release.apk`
- 已签名，可直接提交到应用商店
- 签名文件: `keystore/sat-release.jks`

---

## 二、微信/QQ 发送 APK

### 方法 1: 直接发送文件
1. 打开微信/QQ → 选择联系人
2. 点击"文件" → 选择 `SAT-release.apk`
3. 发送后，对方保存并安装
4. **注意**: 微信可能将 .apk 后缀改为 .apk.1，对方需重命名为 .apk 后安装

### 方法 2: 压缩包发送（推荐）
1. 将 APK 打包为 ZIP: 右键 → 压缩 → SAT.zip
2. 微信发送 ZIP 文件
3. 对方下载后解压，点击 APK 安装

### 用户安装步骤:
1. 下载 APK 文件
2. 点击安装
3. 如弹出"禁止安装未知应用"，前往 设置→安全→允许安装未知应用
4. 安装完成，打开使用

## 三、各大应用商店上架流程

### 1. 华为应用市场 (Huawei AppGallery)
- 注册地址: https://developer.huawei.com
- 开发者类型: 个人/企业（个人需要身份证实名认证）
- 注册费用: 免费（个人）/ 免费（企业需营业执照）
- 审核周期: 3-7个工作日
- APK要求: Release签名版本
- 准备材料:
  * 应用名称: SAT - 智能时间调配
  * 应用分类: 工具 → 效率
  * 应用介绍（120字以内）
  * 应用截图: 4-6张，1080x1920
  * 应用图标: 512x512
  * 《用户协议与隐私政策》
  * 包名: com.sat.timemanager
  * 版本号: 1.0.0
  * ICP备案号（如有）

## 六、更新 APK 流程

每次更新版本时:
1. 修改 `android/app/build.gradle` 中的 versionCode（+1）和 versionName
2. 重新构建 Release APK
3. 在各应用商店提交新版本
4. 更新说明填写变更内容
